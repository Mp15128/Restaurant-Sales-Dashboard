 🍕 Pizza Paradise: Restaurant Sales Dashboard

📊 Project Overview
This project simulates an end-to-end data analysis pipeline for a fictional multi-branch pizza chain called **Pizza Paradise**. Using Python, Pandas, Matplotlib, and Seaborn, we analyze **1,000 unique customer orders** across **10 city branches** over a full calendar year to uncover key performance indicators (KPIs) and operational trends.
